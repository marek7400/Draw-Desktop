The VirusTotal website and the antivirus program on your computer may detect (FALSE) the file as a virus.

This is due, among other things, to the libraries used and the “hook” tools used by the program to operate keyboard shortcuts. 

If anyone is concerned about the exe - let them download the pure PYTHON code. You can check the code yourself and turn it into an exe or run the script itself (as someone has Python installed).

There is an option to report that it's a false alarm, but you have to write to each antivirus vendor separately (and I don't want to).
**************************************************
Witryna VirusTotal i program antywirusowy na komputerze mogą wykryć (FAŁSZYWIE) plik jako wirusa.

Wynika to między innymi z użytych bibliotek i narzędzi „hook” używanych przez program do działania skrótów klawiaturowych. 

Jak ktos się obawia exe - niech ściąga czysty kod PYTHON. Można samemu sprawdzić kod i przekształcić go w exe lub uruchomić sam skrypt (jak ktoś ma zainstalowanego Pythona).

Jest możliwość zgłoszenia, że to fałszywy alarm, ale trzeba pisać do każdego dostawcy antywirusa z osobna (a mi się nie chce).
